# 📊 Placement Dashboard

## Description
This project is a College Placement Dashboard that shows:
- Total Students
- Total Companies
- Placed Students
- Highest Package
- Charts for analysis

## Technologies Used
- HTML, CSS, Chart.js
- Node.js (Express)
- MySQL Database

## How to Run

1. Import PlacementDB.sql into MySQL
2. Run:
   npm install
   node server.js
3. Open index.html in browser

## Features
- KPI Dashboard
- Placement Status Chart
- Branch-wise Analysis
- Company-wise Placement
- Placement Trends
